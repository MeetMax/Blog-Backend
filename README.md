# 博客后台管理系统
> 本项目是一个基于Element UI组件的后台管理系统

## 项目搭建

``` bash
# 克隆项目到本地
git clone https://github.com/MeetMax/Blog-Backend.git

# 进入项目目录
cd Blog-Backend

# 安装依赖
npm install

# 开启本地服务
npm run dev
```
>注意：本项目需要配合后端API使用，关于后端请查看[Blog-API](https://github.com/MeetMax/Blog-API)

## 主要技术栈
- vue
- Element UI
- axios
- webpack
- babel
- simpleMD

## 主要功能
- 发布MarkDown格式的文章
- 管理员进行token 验证登录
- 文章分类的增删改查
